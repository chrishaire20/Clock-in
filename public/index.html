💾 Data Backup

🛡️ 3-Layer Data Protection

Your data is protected three ways simultaneously:

  • Layer 1 — PostgreSQL Database — Render keeps your data even through restarts and deploys
  • Layer 2 — Weekly Email Backup — Full JSON + report emailed to you every Sunday 11pm
  • Layer 3 — Daily Google Drive — Full JSON + CSV uploaded to your Drive folder every night at 2am

☁️ Google Drive — Daily Automatic Backup

Every night at 2am, the system uploads two files to your Google Drive folder:

  • woods-landing-fullbackup-DATE.json — complete database snapshot
  • woods-landing-daily-DATE.csv — that day's clock-ins and outs

One-time setup (5 minutes):

  1. Go to console.cloud.google.com → New Project
  2. Enable Google Drive API
  3. Go to IAM → Service Accounts → Create → Download JSON key
  4. Create a folder in your Google Drive, share it with the service account email
  5. In Render environment variables, add:
      GOOGLE_SERVICE_ACCOUNT_JSON = paste the entire JSON key file content
      GOOGLE_DRIVE_FOLDER_ID = the ID at the end of your folder's URL
